# 🎲 Dice Battle Arena

> A sleek, head-to-head dice game where two players battle to reach 5 points. Built with HTML, CSS, and JavaScript.

[![Live Demo](https://img.shields.io/badge/Live_Demo-Play_Now-brightgreen?style=for-the-badge&logo=githubpages)](https://mushrifa-muzammil.github.io/Dicee_Game/)

---

## 📖 Overview

**Dice Battle Arena** is a two-player dice battle game with a gladiator-themed interface. Each round, both players roll a die. The higher roll wins the round and earns one point. First to **5 points** claims victory.

The game features:
- Clean, dark-themed arena layout
- Dice roll animations with shake effect
- Keyboard shortcuts for fast play
- Real-time score tracking and progress bar
- Winner announcement modal with celebration
- Fully responsive design (laptop, tablet, mobile)

---

## 🎮 Game Rules

| Rule | Description |
|:-----|:------------|
| **Win a Round** | Roll higher than your opponent → gain **+1 point** |
| **Draw** | Both dice show the same number → **no points** |
| **Win the Game** | First player to reach **5 points** wins |
| **Reset** | Press RESET or `ESC` to start a new match |
